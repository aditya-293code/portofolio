# Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript featuring dark/light mode toggle.

## Features

- **Dark/Light Mode Toggle**: Switch between dark and light themes with smooth transitions
- **Responsive Design**: Mobile-first approach that works on all devices
- **Smooth Animations**: Fade-in effects, hover animations, and skill bar animations
- **Interactive Elements**: Hover effects, smooth scrolling navigation
- **Contact Form**: Functional contact form with validation
- **Mobile Menu**: Collapsible navigation for mobile devices
- **Scroll Effects**: Dynamic navbar styling on scroll

## Technologies Used

- HTML5
- CSS3 (Custom Properties for theming, Flexbox, Grid, CSS Variables)
- JavaScript (ES6+ features, localStorage for theme persistence)
- Font Awesome Icons
- Google Fonts (Poppins)
- Responsive design with CSS Grid and Flexbox

## Sections

1. **Hero**: Introduction with profile image and call-to-action buttons
2. **About**: Personal information and background
3. **Skills**: Technical skills with animated progress bars
4. **Projects**: Showcase of completed projects with hover effects
5. **Contact**: Contact information and functional form
6. **Footer**: Social links and copyright

## Theme System

The website uses CSS custom properties (variables) for easy theming:

- Light theme: Clean, bright design
- Dark theme: Modern dark interface
- Automatic theme persistence using localStorage
- Smooth transitions between themes

## How to Run

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Or use a local server:
   ```bash
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000`

## Customization

- Update personal information in `index.html`
- Modify colors in CSS custom properties at the top of `style.css`
- Add your own projects and images
- Update social media links in the footer
- Customize theme colors by editing the CSS variables
